# AI Doctor Recommendation — User Guide

## What Is "Find by Symptoms"?

"Find by Symptoms" is a tool in the patient sidebar that helps you identify which type of doctor to see based on what you're experiencing. You describe your symptoms, and the tool suggests the appropriate medical specialization.

---

## How to Use It

1. Click **Find by Symptoms** in the left sidebar (marked with a sparkle icon)
2. Click any symptoms that apply to you — they turn blue when selected
3. Click **Get Recommendation**
4. The tool shows you the recommended specialization(s) and a link to browse matching doctors

---

## Symptom List

The tool includes common symptoms across major medical specializations:

| Symptom | Recommended Specialization(s) |
|---|---|
| Chest pain | Cardiology, Internal Medicine |
| Heart palpitations | Cardiology |
| Shortness of breath | Pulmonology, Cardiology |
| Persistent cough | Pulmonology, ENT |
| Skin rash / acne | Dermatology |
| Headache | Neurology, Internal Medicine |
| Dizziness | Neurology, ENT |
| Back pain | Orthopedics, Internal Medicine |
| Joint pain | Orthopedics, Rheumatology |
| Fever | Internal Medicine, Infectious Disease |
| Stomach pain / nausea | Gastroenterology |
| Eye problems | Ophthalmology |
| Ear pain / runny nose | ENT, Allergy |
| Anxiety / depression | Psychiatry, Psychology |
| Fatigue / weight gain | Endocrinology, Internal Medicine |
| Women's health | OB-GYN |
| Urinary problems | Urology |

You can select multiple symptoms — the tool combines the recommendations for all selected symptoms.

---

## After Getting a Recommendation

The result panel shows:
- A list of recommended specializations as badges
- A **Browse [Specialization] Doctors** button that links directly to the doctor discovery page filtered by that specialization

## Important Disclaimer

This tool is a **general guidance aid only**. It is not a substitute for professional medical advice, diagnosis, or treatment. The recommendations are based on a static mapping and do not account for your full medical history.

**In an emergency**, call emergency services (911 or your local emergency number) immediately — do not use this tool.

---

## Clearing and Starting Over

Click **Clear** to deselect all symptoms and start a new symptom search.
